<h2 id="execution">👨🏻‍💻 Execution</h2>

To run the project:

- Clone the project by running this command in the terminal `git clone <repo_url>`;
- Open the backend folder in the terminal:

  - Run with docker ([Click here to install docker](https://docs.docker.com/engine/install/)):
    - linux: `docker compose up`
    - windows: `docker-compose up`

- Open the frontend folder in the terminal:
  - `yarn` or `npm install` -> to install dependencies
  - `yarn dev` or `npm run dev` -> to run project
